# torrent-search-go - Marketing Site

Static assets for the GitHub Pages landing page at
`https://akshatsinghkaushik.github.io/torrent-search-go/`.

## Files

| File | Description |
|------|-------------|
| `favicon.svg` | SVG favicon (dark background, cyan clock icon) |
| `og-image.svg` | 1200×630 Open Graph image for social sharing |
| `robots.txt` | Search engine crawl directives |
| `sitemap.xml` | XML sitemap |
| `site.webmanifest` | Web app manifest |

## How GitHub Pages Works

GitHub Pages is configured to serve the `docs/` directory. The landing page
lives at `docs/index.html`. The `.nojekyll` file in `docs/` tells GitHub Pages
not to run Jekyll processing, so all files are served as-is.

No build step is required - the page uses Tailwind CSS via Play CDN and
vanilla JavaScript only.
